## 1. Профиль использования бота

| Параметр | Значение |
|----------|----------|
| System prompt | ~3,500 tokens (сложные инструкции + техники) |
| RAG контекст | ~500-1000 tokens на запрос |
| Средний запрос пользователя | ~100-300 tokens |
| Средний ответ | ~150-400 tokens (3-4 предложения) |
| Глубокий анализ (ассоциации/история) | ~800-1500 tokens |
| Температура | 0.7-0.8 (требуется креативность) |
| Языки | Русский, Английский |

**Типичные сценарии:**
- **Короткий обмен:** 500 input / 200 output tokens
- **Средняя сессия:** 2000 input / 400 output tokens  
- **Глубокий анализ:** 3000 input / 800 output tokens

## 7. ✅ Реализованная стратегия

### Гибридный подход (уже внедрён в `src/therapist_bot.py`):

```python
# Конфигурация по умолчанию
self.model = "gpt-4o-mini"              # Для обычного чата
self.analysis_model = "claude-3-opus-latest"  # Для глубоких анализов
```

**Использование моделей:**
| Функция | Модель | Параметр |
|---------|--------|----------|
| `chat()`, `chat_stream()`, `generate_response()` | GPT-4o-mini | `self.model` |
| `analyze_associations()` | Claude-3-Opus | `self.analysis_model` |
| `analyze_story()` | Claude-3-Opus | `self.analysis_model` |

### CLI аргументы:
```bash
# По умолчанию (рекомендуется)
python src/therapist_bot.py

# Кастомные модели
python src/therapist_bot.py --model gpt-4o-mini --analysis-model claude-3-opus-latest
```

## 8. Технические замечания

### Поддержка провайдеров в коде:

```python
# Текущая реализация (src/therapist_bot.py)
self.model = "claude-3-5-haiku-latest"  # или "gpt-4o-mini", "llama-3.1-70b"

# Рекомендуемые настройки для разных моделей:
MODEL_CONFIGS = {
    "gpt-4o-mini": {
        "temperature": 0.7,
        "max_tokens": 1000,
        "cost_per_1k_input": 0.00015,
        "cost_per_1k_output": 0.00060,
    },
    "gemini-2.0-flash": {
        "temperature": 0.8,
        "max_tokens": 1000,
        "cost_per_1k_input": 0.00010,
        "cost_per_1k_output": 0.00040,
    },
    "claude-3-opus-latest": {
        "temperature": 0.7,
        "max_tokens": 1500,
        "cost_per_1k_input": 0.015,
        "cost_per_1k_output": 0.075,
    }
}
```

### Важно для existential-бота:

1. **Temperature:** 0.7-0.8 оптимально (нужна креативность для метафор, но не галлюцинации)
2. **Max tokens:** 1000 для обычных ответов, 1500 для анализов
3. **System prompt:** Все модели поддерживают 3500+ tokens (кроме возможных ограничений glm-4.6)
4. **Streaming:** Поддерживается всеми моделями через OpenAI-compatible API
